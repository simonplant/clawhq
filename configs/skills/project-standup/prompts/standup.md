# Standup Composition

Format: {{format_style}}

Yesterday completed:
{{completed_tasks}}

Today planned:
{{todays_tasks}}

Blocked:
{{blocked_tasks}}

Write a standup update. Rules:
- Specific, not vague. "Shipped auth flow" not "worked on auth."
- If nothing completed: "Nothing shipped yesterday." Don't fill with process words.
- Blockers: one line each. What it is and what's needed to unblock.
- Total length: 100-200 words max.
- Tone: direct, peer-to-peer. Not a status report to a boss.
