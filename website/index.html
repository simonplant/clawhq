Get started in one command

Linux, macOS, or WSL. Your PC, a Mac Mini, or any VPS.

Trusted install Recommended
curl -fsSL https://clawhq.com/install | sh

Signed, hash-verified. Requires curl + Docker.

Zero-trust install
git clone https://github.com/simonplant/clawhq
cd clawhq && ./install --from-source --verify

Clone, audit, build from source. For the security-paranoid (no judgment — we'd do the same).

Then:

1
clawhq init --guided

Pick a blueprint, connect your services, customize.

2
clawhq up

Preflight checks, hardened container, smoke test, firewall.

3
🐾

Your agent is running. Say hello on Telegram.
